Introducció Completa a p5.js

Explora el poder de p5.js per crear art interactiu, gràfics i molt més.

Què és p5.js?

p5.js és una biblioteca de JavaScript que facilita la programació creativa i el treball amb gràfics, so i més. A continuació, et mostrem un exemple de com crear un llenç i dibuixar un cercle.

Exemple Bàsic:

function setup() {
  createCanvas(400, 400); // Crear el llenç
}

function draw() {
  background(220); // Fons gris clar
  ellipse(200, 200, 50, 50); // Dibuixa un cercle
}

Aquest codi genera un llenç de 400x400 píxels i dibuixa un cercle al centre. Fàcil, oi?

Estructura bàsica

A p5.js, sempre trobaràs dues funcions principals: setup() i draw().

function setup() {
  createCanvas(400, 400); // Llenç inicial
}

function draw() {
  background(220); // Fons que s'actualitza contínuament
}

setup() s'executa una sola vegada al principi, i draw() s'executa repetidament per actualitzar la pantalla.

Treballant amb variables

Les variables s'utilitzen per emmagatzemar valors que poden canviar durant l'execució del programa. En el següent exemple, controlem la posició d'un cercle utilitzant variables.

let x = 200;
let y = 200;

function draw() {
  background(220);
  ellipse(x, y, 50, 50); // Dibuixa el cercle a la posició (x, y)
  x += 2; // Mou el cercle cap a la dreta
}

El valor de x augmenta a cada fotograma, fent que el cercle es desplaci horitzontalment.

Ús de funcions

Les funcions permeten organitzar el codi i reutilitzar-lo de manera més eficient. A continuació es mostra com crear una funció per dibuixar cercles en diferents posicions.

function dibuixaCercle(x, y) {
  ellipse(x, y, 50, 50); // Dibuixa el cercle a la posició (x, y)
}

function draw() {
  background(220);
  dibuixaCercle(100, 100);
  dibuixaCercle(300, 300);
}

Aquest codi dibuixa dos cercles en posicions diferents al mateix temps.

Exemples interactius

Exemple 1: Seguir el ratolí

function setup() {
  createCanvas(400, 400);
}

function draw() {
  background(220);
  ellipse(mouseX, mouseY, 50, 50); // Dibuixa un cercle a la posició del ratolí
}

Aquest codi dibuixa un cercle que segueix el moviment del ratolí a la pantalla.

Exemple 2: Moviment d'un cercle

let x = 0;
function setup() {
  createCanvas(400, 400);
}

function draw() {
  background(220);
  ellipse(x, height / 2, 50, 50);
  x += 2;
  if (x > width) {
    x = 0;
  }
}

Aquest codi mou un cercle horitzontalment per la pantalla i el reinicia quan arriba a la vora.
